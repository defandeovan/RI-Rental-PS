# RentalPS

## Deskripsi Aplikasi
RentalPS adalah aplikasi manajemen rental PlayStation yang dirancang untuk memudahkan pemilik usaha rental PS dalam mengelola peminjaman konsol, pengaturan jadwal, dan pencatatan transaksi. Aplikasi ini menyediakan fitur monitoring ketersediaan konsol, manajemen pelanggan, serta laporan keuangan yang membantu meningkatkan efisiensi operasional bisnis rental PlayStation.
